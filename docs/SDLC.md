# Software Development Life Cycle (SDLC)

## Requirement Analysis
Requirements were gathered and documented in the SRS document.

## System Design
UML diagrams were used to model system behavior and structure.

## Implementation
The system is implemented using Spring Boot and React following Agile practices.

## Testing
Basic functional testing will be performed for core features.

## Deployment
The application will be containerized using Docker.

## Maintenance
Future improvements and bug fixes will be managed through issue tracking.
